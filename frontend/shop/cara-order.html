1. Syarat & Ketentuan

  • Usia minimal 17 tahun (atau sesuai hukum yurisdiksi kamu)
  • Email aktif untuk verifikasi akun + terima kredensial produk
  • Akun yang kamu beli hanya untuk penggunaan pribadi/non-komersial kecuali dinyatakan commercial license
  • Dilarang memperjualbelikan kembali kredensial yang dibeli di Z Store
  • Untuk produk dengan commercial license (Midjourney Pro, ChatGPT Team, dll), boleh dipakai untuk deliverable klien
  • Dengan order, kamu setuju dengan Terms of Service + Refund Policy

2. Lima Langkah Order

1
Daftar / Login Buat akun via email + password (wajib OTP verify) ATAU login cepat via Google / Telegram / Discord / WhatsApp. Setelah login, kamu bisa link semua platform jadi satu akun.
2
Pilih Produk Browse di homepage atau klik kategori. Setiap produk ada detail: harga, garansi, format (akun | voucher | API key | joki), dan reviews dari buyer lain.
3
Add to Cart & Checkout Klik "CART" untuk masukkan ke keranjang, atau "BELI" untuk langsung checkout. Review pesanan, masukkan kode promo kalau ada (mis. WELCOME50 untuk diskon Rp 50k), klik Checkout.
4
Bayar via Midtrans Pilih metode pembayaran favorit (QRIS / Virtual Account / GoPay / ShopeePay / Credit Card). Pembayaran diproses real-time. Setelah bayar, status berubah ke paid otomatis.
5
Terima & Konfirmasi Kredensial produk dikirim ke email kamu dalam <10 menit. Cek inbox + spam folder. Login ke produk, test apakah jalan. Kalau ada masalah, klik "Konfirmasi Terima" di halaman order setelah 24 jam (escrow release).

4. Delivery & Akses Produk

Format Pengiriman

  • Akun (email + password) → Dikirim via email + tampil di dashboard Orders
  • Voucher / License Key → Kode unik dikirim via email. Redeem di website resmi produk
  • API Key → String key + dokumentasi cara pakai
  • Joki Service → Tim kami akan request akses (opsional) + deliver hasil via Dashboard / Google Drive
  • Source Code → Download link (Google Drive / GitHub Release)

Waktu Pengiriman

Target Delivery: <10 menit setelah pembayaran confirmed. Kalau lebih dari 30 menit belum terima, hubungi WhatsApp CS. Kemungkinan penyebab: email salah ketik, inbox spam, atau delay stok.

Test & Konfirmasi

Setelah terima kredensial:

  1. Login ke produk dengan kredensial yang diberikan
  2. Test semua fitur utama (jangan ubah password sebelum konfirmasi)
  3. Kalau jalan → klik Konfirmasi Terima di halaman order (escrow released ke seller)
  4. Kalau ada masalah → klik Bantuan di halaman order untuk komplain

5. Sistem Escrow & Garansi

Perlindungan Escrow 7 Hari

Setelah pembayaran, dana ditahan di escrow (bukan langsung diteruskan ke seller). Hanya released kalau:

  • Kamu klik Konfirmasi Terima (release instant)
  • Lewat 7 hari tanpa komplain (auto-release)

Garansi 30 Hari

Semua produk digital bergaransi 30 hari. Kalau:

  • Akun tiba-tiba tidak bisa login (credential invalid / banned)
  • Voucher sudah redeemed sebelumnya (double-use)
  • API key mati sebelum masa aktif
  • Joki tidak deliver hasil sesuai pesanan

→ Kamu berhak dapat replacement baru atau refund 100%. Buka tiket di Bantuan.

Penting: Jangan ganti password / logout dari produk sebelum konfirmasi terima. Kalau ganti password dan ternyata akun invalid, kami tidak bisa verifikasi keasliannya.

7. FAQ Singkat

Q: Berapa lama kredensial dikirim?

A: <10 menit setelah pembayaran confirmed. Via email + dashboard.

Q: Bisa order tanpa akun?

A: Bisa via Guest Checkout, tapi lebih cepat kalau punya akun (riwayat order tersimpan, tidak perlu input alamat berulang).

Q: Bisa refund?

A: Bisa dalam 24 jam pertama selama kredensial belum digunakan. Setelah 24 jam atau setelah dikonfirmasi, garansi 30 hari via replacement. Lihat Refund Policy.

Q: Bisa order via WhatsApp / Telegram / Discord?

A: Bisa. Bot Z Store di platform tersebut akan kirim katalog + link checkout. Atau langsung balas pesan dengan kode produk, CS yang proses manual.

Q: Gimana cara joki service?

A: Order produk "Joki [nama game/service]" → setelah bayar, tim akan DM kamu via WhatsApp/Telegram untuk request akses + detail task. Deliver dalam 1-7 hari tergantung kompleksitas.
