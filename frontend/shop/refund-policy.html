Kebijakan Refund

Transparan dan fair. Refund 100% dalam 24 jam pertama jika produk tidak terkirim / tidak sesuai deskripsi. Setelah itu, garansi 30 hari via replacement.

Window Refund

Waktu Sejak OrderHak Refund
0-24 jam (belum terima kredensial)100% refund (wajib — stok habis / gagal deliver)
0-24 jam (sudah terima, produk tidak jalan)100% refund atau replacement baru (pilihan kamu)
1-7 hari (produk jalan, berubah pikiran)Tidak ada refund, tapi replacement tersedia kalau produk rusak dari sisi kami
7-30 hari (produk tiba-tiba rusak / banned)Replacement baru via garansi 30 hari. Refund 50% sebagai alternatif.
>30 hariTidak ada refund / replacement. Terima kasih sudah jadi customer.
Pola Umum: 24 jam pertama = refund 100% jika produk gagal deliver. Setelahnya = garansi replacement (bukan refund tunai).

Yang Bisa Di-Refund

  • Produk tidak terkirim dalam 30 menit setelah pembayaran
  • Kredensial tidak bisa login (invalid / banned) padahal baru diberikan
  • Voucher / API key sudah pernah dipakai orang lain (double-use)
  • Source code corrupt / tidak bisa dijalankan
  • Joki service gagal deliver sama sekali
  • Deskripsi produk tidak sesuai dengan fitur yang diterima (mis. dijanjikan "Pro Plan" ternyata "Free Plan")

Yang TIDAK Bisa Di-Refund

  • Refund karena kamu tidak jadi pakai / berubah pikiran setelah pakai
  • Refund karena kesalahan input dari kamu (order barang salah, salah platform)
  • Akun yang di-ban karena aktivitas kamu sendiri (spam, scam, ToS violation)
  • Refund setelah kamu konfirmasi terima (artinya kamu sudah setuju)
  • Refund karena koneksi internet lambat di device kamu
  • Refund karena kamu merasa "kemahalan" setelah pakai (no buyer's remorse after use)

Proses Refund

  1. Buka halaman Orders → klik order → Komplain / Refund
  2. Pilih alasan refund + lampirkan bukti
  3. Tim CS review dalam 1 jam (jam kerja 09:00-22:00 WIB)
  4. Setelah disetujui, refund diproses ke metode pembayaran asal
  5. Waktu pencairan: 3-7 hari kerja (tergantung bank / e-wallet)
Catatan: Refund tidak bisa ke metode pembayaran lain. Kalau order via QRIS GoPay, refund masuk ke GoPay (bukan ke rekening bank).

Biaya Admin

Refund dipotong biaya admin payment gateway (Midtrans) yang sudah kami bayar di awal:

  • QRIS / E-Wallet: 0.7% dari nominal refund
  • Virtual Account: Rp 4.000 / transaksi
  • Credit Card: 2.9% dari nominal refund

Biaya ini dipotong dari nominal refund (jadi kamu terima refund = nominal order - biaya admin). Contoh: order Rp 100.000 via QRIS → refund Rp 99.300.
